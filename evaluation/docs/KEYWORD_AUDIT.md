# 키워드 등장 감사 — 결과 분석

신규 사전 entry 추가 의사결정의 근거. **추가 후보 ~85개를 실제 데이터(keywords.parquet + papers.논문명) 에서 substring 매칭** 한 결과.

데이터 파일: [`../output/keyword_audit.csv`](../output/keyword_audit.csv)
스크립트: [`../labeling/audit_keyword_coverage.py`](../labeling/audit_keyword_coverage.py) — 멱등 재실행 가능

판정 기준:
- **INCLUDE**: kw + title 합산 ≥ 3 → 사전 entry 추가
- **MARGINAL**: 합산 1~2 → 사전 추가 + 검수 표본 자동 포함
- **SKIP**: 0 → 사전 추가 안 함 (등장 시 재고)

## 2. 핵심 발견과 의미

### 2.1 일본 불교 부재 — 그 자체가 결과

**親鸞·道元·空海 전부 0건.** 단일 변형도 없음.
일본 불교의 대표 종파(정토진종·조동선·진언종) 인물이 키워드·제목에 한 번도 등장 안 함.

**의미**:
- `<인도철학>` 학술지는 "동아시아 불교" 를 다룬다 해도 일본 불교는 사실상 사각지대.
- 축 1 (커버리지) 결과 보고 시 "일본 불교 비중 ≈ 0%" 로 *부재의 정량화* 가능.
- 축 6 (학제 경계) 에서 `reception_horizon=east_asia` 의 내적 구성을 보면 중국·한국 편중이 강하게 드러날 전망.

### 2.2 한국 불교 = 원효 압도

원효 키워드 7건 + 제목 3건 = **10건**.
의상·지눌·휴정·서산·의천·균여·함허·진감·대각 **모두 0건**.

**의미**: 인도철학 학술지 안에서 "한국 불교" 는 사실상 *원효 1인의 사상*.
- 지눌의 돈오점수, 의상의 화엄, 의천의 천태 등 한국 불교사 주요 인물이 부재.
- 한국 불교사 일반은 한국불교학회·불교학연구·보조사상 등 인접 학술지의 영역이라 추정 (축 4 단계에서 검증).

### 2.3 중국 화엄·천태의 *학파명은 있고 인물은 없음*

| | 학파명 | 인물명 |
|---|---|---|
| 화엄 | 3건 (kw2+title1) | 法藏·智儼·澄觀 = 0건 |
| 천태 | 0건 (한글) — 한자 "天台" 도 0건 | 智顗 (한자) 0건, "지의" (한글) 3건 |
| 정토 | 11건 (kw7+title4) | 善導·法然 = 0건 |
| 법상 | 1건 (제목) | 窺基 (한자) 0건, "규기" 3건 |
| 진언 | 17건 | 空海 = 0건 |

**의미**: 한국 학자들은 동아시아 종파를 *학설·교리 단위* 로 다루지 *역사 인물 단위* 로 다루지 않음.
- 추론: 인도철학 학술지 = "교리·사상" 중심, "역사·인물 전기" 는 인접 학술지의 몫?
- 검증 방법: 축 4 (인접 학술지 비교) 에서 "법장·지엄·혜능" 키워드 빈도 비교.

### 2.4 천태와 법상의 한자 표기 부재

"天台" 0건, "法相" 0건, "禪宗" 0건 — **학파의 한자 정식 명칭이 키워드에 없음**.

대신:
- 천태 사상은 `지의(智顗)` 3건 + `법화경` 4건 으로 *간접 추정*
- 법상 사상은 `규기` 3건 + `원측` 3건 + `법상` (제목) 1건 으로 *추정*

**라벨링 함의**: 학파명 단독으로 신호가 약하므로, **인물·텍스트 entry 가 더 결정적 신호**.

### 2.5 미맘사 — 사전 누락 발견

미맘사 (kw1 + title2 = 3건) → INCLUDE.
인도 정통 6파 중 우리 사전에 빠져 있었음. **사전 보완**.

### 2.6 요가 학파의 era 다양성 데이터에 실재

| era 변형 | 빈도 |
|---|---|
| "8지 요가" / "고전 요가" (= classical) | 키워드에 등장 |
| "하타 요가" / "하타요가" / "하타프라디피카" (= post_classical) | kw9+title6 |

**의미**: Decision-08 (yoga era 필수) 의 근거가 데이터로 확증됨. 하타 요가는 별도 entry 가치 충분.

### 2.7 자료·방법 키워드의 강한 빈도

| 키워드 | 빈도 | 의미 |
|---|---|---|
| 사본 | 17+8 | 사본학 방법론이 학술지의 한 흐름 |
| 돈황 | 5+5 | 동아시아 자료 + 인도불교 한역의 핵심 |
| 한역 | 3+4 | `source_language=chinese` 추론에 활용 |
| 위빠사나 | 8+2 | 빠알리 불교 명상 — abhidharma + pali source |
| bhakti | 0+4 (제목에만) | 박티 운동 — `school=hindu_modern` 의 신호 |

→ 자료·방법 키워드는 *학파* 보다 *언어/지평/시대* 의 신호로 활용 가능.
